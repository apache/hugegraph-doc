Metrics API

HugeGraph 提供了获取监控信息的 Metrics 接口，比如各个 Gremlin 执行时间的统计、缓存的占用大小等。Metrics 接口包括如下几类：基础指标、统计指标、系统指标、后端存储指标。

1. 基础指标​

1.1 获取所有基础指标​

Params​
  • type：如果传值为json，则以 json 格式返回，否则以 Promethaus 格式返回。
1.1.1 Method & Url​
http://localhost:8080/metrics/?type=json
